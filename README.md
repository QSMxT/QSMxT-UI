# QSMxT-UI

# Dependencies

## Build dependencies

- nodejs

  You can install from https://nodejs.org/en/download/

- yarn

Install using

```bash
npm install --global yarn
```

```bash
$ node --version
v12.13.0
$ npm --version
6.12.0
$ yarn --version
1.22.19
```

## sqlite3

```bash
sudo apt-get install libsqlite3-dev
npm install sqlite3 --build-from-source
```

# Install

`npm install`

# Run

```bash
cd frontend
npm run start
```
